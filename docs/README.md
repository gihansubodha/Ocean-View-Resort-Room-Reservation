# Documentation README

This folder stores the main supporting documents used for the assignment.

## Subfolders
- `uml/` - UML diagrams such as use case, class, and sequence diagrams
- `testing/` - test evidence, automation outputs, and QA materials
- `screenshots/` - UI, GitHub, and deployment screenshots used in the report
- `decisions/` - design decision notes and justifications
- `unified-process/` - artefacts related to planning and development phases

## Purpose
These files support the written report, viva preparation, and evidence trail for the system design and implementation.
